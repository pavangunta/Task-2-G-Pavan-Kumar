# Iris Flower Classification using K-Nearest Neighbors (KNN)

## Overview

This project is an interactive implementation of the K-Nearest Neighbors (KNN) classification algorithm using the famous Iris Dataset.

Users can enter flower measurements and adjust the value of K to classify an Iris flower as:

* Setosa
* Versicolor
* Virginica

The project demonstrates the core concepts of supervised machine learning and classification.

## Features

* Interactive KNN classifier
* Adjustable K value
* Real-time predictions
* Neighbor voting visualization
* Preset sample flowers
* Clean and responsive UI

## Technologies Used

* React.js
* JavaScript
* K-Nearest Neighbors (KNN)

## Dataset

The Iris dataset contains:

* 150 flower samples
* 3 species
* 4 features:

  * Sepal Length
  * Sepal Width
  * Petal Length
  * Petal Width

## Algorithm

KNN classifies a flower by:

1. Calculating distances from the input sample to all training samples.
2. Selecting the K nearest neighbors.
3. Performing majority voting.
4. Returning the predicted flower species.
